The full form of DOM is Document Object Model. It uses JavaScript to Change and Create the elements, attributes of html & CSS. It can be used to create a Dynamic website and adds functionality to interact with the website

using "getElementById" , "getElementByClassName" & "getElementByTagName"

Event Bubbling is a concept in DOM. It happens when an element receives an event, and that event bubbles up or in other words it is transmitted to its parent and previous elements in DOM tree and it happens until it reaches to the root element.

CallBack function is a function where another function is called as a parameter. Or a function can be called inside another function.
